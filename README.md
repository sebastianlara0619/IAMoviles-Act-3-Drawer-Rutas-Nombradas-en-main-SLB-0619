# myapp

# Prompt
Actúa como un desarrollador experto en Flutter. Crea el código completo de una app llamada 'Celulares Lara' que use Rutas Nombradas para la navegación.

Requerimientos Técnicos:

Drawer (Menú Lateral): Debe tener un encabezado (UserAccountsDrawerHeader) con una imagen de avatar desde la red, el nombre de la empresa, dirección, teléfono y correo electrónico.

Opciones del Menú: Incluye 4 opciones relacionadas con la venta de celulares (Catálogo, Sucursales, Empleados y Soporte Técnico) usando ListTile con iconos temáticos y acciones para navegar a sus respectivas páginas.

Vistas de Contenido: Cada página debe mostrar su título y una imagen central de 200x200 píxeles cargada desde un repositorio de GitHub.

Estética: Aplica un tema visual minimalista y tecnológico (usando colores como Indigo o BlueGrey) acorde a una tienda de smartphones.

Entorno: El código debe ser compatible con Dart y estar preparado para integrarse con Firebase en el futuro.

Extra: Por favor, genera también un logo minimalista y elegante para la marca que combine un smartphone con la letra 'L'.
